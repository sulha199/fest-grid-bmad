# 00.2-logout

## Page Metadata

| Property | Value |
|---|---|
| **Scenario** | 00-Login-Flow |
| **Page Number** | 00.2 |
| **Platform** | Web (Desktop) |

## Overview

**Page Purpose:** To allow a user to securely log out of their account.

**Entry Context:** A logged-in user clicks the "Logout" button, typically found in the user menu.

**Exit Action:** The user is logged out and redirected to the home page.

## User Situation

"Chloe" has finished her session and wants to log out of her FestGrid account.

## Mental State

-   **Security-conscious:** Chloe wants to ensure her account is secure after she has finished using the application.

## On-Page Interactions

1.  Chloe clicks on the user menu to reveal a list of options.
2.  She clicks on the "Logout" button.
3.  The system logs her out, invalidating her session.
4.  Chloe is redirected to the FestGrid home page.
5.  A confirmation message (e.g., a toast notification) appears briefly, saying "You have been logged out successfully."
