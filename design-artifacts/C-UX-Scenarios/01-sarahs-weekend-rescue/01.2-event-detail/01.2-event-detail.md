## Overview

**Page Purpose:**
To get all the necessary information to feel confident about attending an event. This includes: event types and categories (displayed as tags), schedule, performers, price, location (with map link), link to original post, and contact person.

**Entry Context:**
User arrives from the 'Home/Event Discovery' page after clicking on an event.

**Exit Action:**
Clicks the "Add to Calendar" button.

**On-Page Interactions:**
- View event details, including type and category tags.
- Click link to map.
- Click link to original post.
- Favorite/Bookmark the event.
