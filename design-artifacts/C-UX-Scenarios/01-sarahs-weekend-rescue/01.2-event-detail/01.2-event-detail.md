# 01.2-event-detail

## Page Metadata

| Property | Value |
|---|---|
| **Scenario** | 01-Sarahs-Weekend-Rescue |
| **Page Number** | 01.2 |
| **Platform** | Mobile |

## Overview

**Page Purpose:** To view the detailed information for a single event and take action on it (favorite, add to calendar).

**Entry Context:** Sarah has found an interesting event on the discovery page and has tapped on it to learn more.

**Exit Action:** Sarah navigates back to the discovery page or to her "Favorites" or "Calendar" page.

## User Situation

Having found a "Weekend Puppet Workshop" on the discovery page, Sarah wants to see if it's a good fit for her kids. She needs to know the exact times, what it entails, and if there are any costs.

## Mental State

-   **Investigative:** She is actively looking for specific details to help her make a decision.
-   **Cautiously Optimistic:** She's hopeful this is the right event, but needs to confirm the details.

## On-Page Interactions

### Viewing Event Details
- The page displays all the key information for the "Weekend Puppet Workshop":
    - Event Name
    - Description
    - Date(s) and Time(s) for all schedules
    - Location (with a tappable map link)
    - Performers/Organizers
    - Ticket Price
    - Category and Type tags
- Sarah sees that there are two schedules: "Saturday Session (Ages 5-7)" and "Sunday Session (Ages 8-10)". This is perfect for her two kids.

### Favoriting the Event
- Sarah decides she's interested but wants to discuss it with her partner first.
- She taps the "Favorite" (heart) icon at the top of the page. The icon fills in, providing clear visual feedback. Tapping it again would unfavorite the event, and the icon would return to an outline.

### Adding to Calendar
- After discussing it, they decide to go. Sarah returns to the event details page.
- She taps the "Add to Calendar" button.
- A dialog appears, showing the two separate schedules and allowing her to select which ones to add.
    - [ ] Saturday Session (Ages 5-7)
    - [ ] Sunday Session (Ages 8-10)
- She checks both boxes and taps "Confirm".
- The app integrates with her phone's native calendar, adding the two events.
- A confirmation message appears: "Event has been added to your calendar."
- The "Add to Calendar" button now changes to "View in Calendar" or shows a similar "added" state.
