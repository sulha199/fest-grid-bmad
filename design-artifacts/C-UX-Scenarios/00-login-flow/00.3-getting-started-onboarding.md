# 00.3-getting-started-onboarding

## Overview

**Page Purpose:** To provide a smooth onboarding experience for new users, guiding them through the initial setup process.

**Entry Context:** A new user signs up for a FestGrid account.

**Exit Action:** The user completes the onboarding wizard and starts exploring the application.

## User Situation

A new user, "Kevin", has just signed up for FestGrid and is using the application for the first time.

## Mental State

-   **Curious:** Kevin is interested in what FestGrid has to offer.
-   **Open to guidance:** He is looking for a clear and easy way to get started.

## On-Page Interactions

1.  After signing up, Kevin is automatically redirected to the "Getting Started" wizard.
2.  The wizard presents a series of steps, such as:
    -   A brief welcome message and explanation of the app's core features.
    -   A prompt to add his first social media subscription.
    -   Guidance on how to select posts for event extraction.
    -   An optional step to add his Gemini API key, with a clear explanation of the benefits.
3.  Kevin follows the steps in the wizard, providing the necessary information.
4.  At the end of the wizard, he is taken to his personalized feed, which is now populated with events from his first subscription.
