## Overview

**Page Purpose:** To create, view, edit, and delete saved locations for event discovery and notifications.

**Entry Context:** User navigates to the "My Locations" or "Location Preferences" screen from their user profile or settings menu.

**Exit Action:** User navigates away from the screen after successfully managing their locations.

## User Situation

Alex is a new user who wants to set up his important locations in FestGrid to get relevant event recommendations. He wants to add his "Home" and "Work" locations.

## Mental State

-   **Goal-oriented:** Wants to set up the app correctly to get the most value out of it.
-   **Expects it to be simple:** Assumes managing a list of locations will be a straightforward task.

## On-Page Interactions

### Viewing Locations
- Alex sees a list of his currently saved locations. Initially, this list is empty, with a prominent "Add a New Location" button.

### Adding a New Location
- Taps the "Add a New Location" button.
- A new screen or modal appears with the following fields:
    - **Name:** A text input for the location's name (e.g., "Home").
    - **Address/Location:** A search box that uses a map/geolocation service to find the location. Alex can also tap a "Use Current Location" button.
    - **Radius:** A slider or input to set the search radius in kilometers (e.g., 5km).
- Alex types "Home" in the name field.
- He types his home address into the location search box and selects the correct address from the results.
- He leaves the radius at the default of 5km.
- He taps "Save".
- He is returned to the location list and now sees "Home" in the list.

### Editing a Location
- Alex taps on the "Home" location in the list.
- He is taken to the same "edit" screen, with the fields pre-populated.
- He changes the radius from 5km to 10km.
- He taps "Save".
- He is returned to the location list.

### Deleting a Location
- In the location list, Alex swipes left on his "Work" location (which he added previously).
- A "Delete" button appears.
- He taps "Delete".
- A confirmation dialog appears: "Are you sure you want to delete 'Work'?"
- He confirms, and the "Work" location is removed from the list.
